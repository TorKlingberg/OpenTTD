  OpenTTD Airport System Architecture                                                                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                        
  Core Components                                                                                                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                                                                                        
  1. Finite State Machine (FSM) System                                                                                                                                                                                                                                                                                                                                  
  - airport.h/cpp - The heart of airport operations                                                                                                                                                                                                                                                                                                                     
  - Each airport type has an AirportFTAClass defining a complete state machine                                                                                                                                                                                                                                                                                          
  - States represent positions: hangars, terminals, runways, taxiways, holding points                                                                                                                                                                                                                                                                                   
  - AirportFTA structs define transitions between states with movement flags                                                                                                                                                                                                                                                                                            
  - AirportMovingData contains exact x,y,z coordinates for each position                                                                                                                                                                                                                                                                                                
                                                                                                                                                                                                                                                                                                                                                                        
  2. Aircraft Movement                                                                                                                                                                                                                                                                                                                                                  
  - aircraft_cmd.cpp - Implements all aircraft behavior (~2200 lines)                                                                                                                                                                                                                                                                                                   
  - Event handlers for each state: AircraftEventHandler_EnterTerminal, AircraftEventHandler_TakeOff, etc.                                                                                                                                                                                                                                                               
  - Aircraft follow FSM paths through the airport                                                                                                                                                                                                                                                                                                                       
  - Block reservation system prevents collisions (64-bit bitmask)                                                                                                                                                                                                                                                                                                       
                                                                                                                                                                                                                                                                                                                                                                        
  3. Airport Types & Data                                                                                                                                                                                                                                                                                                                                               
  - table/airport_movement.h - Movement data for all default airport types                                                                                                                                                                                                                                                                                              
  - table/airport_defaults.h - Tile layouts and graphics for each airport                                                                                                                                                                                                                                                                                               
  - table/airporttile_ids.h - 256 different airport tile graphics                                                                                                                                                                                                                                                                                                       
  - 9 default airport types: Small, Commuter, City, Metropolitan, International, Intercontinental, Heliport, Helidepot, Helistation                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                                                                        
  4. Station Integration                                                                                                                                                                                                                                                                                                                                                
  - station_base.h - Airport struct attached to Station                                                                                                                                                                                                                                                                                                                 
  - Contains: type, layout, rotation, block occupancy bitmask                                                                                                                                                                                                                                                                                                           
  - AirportTileIterator for iterating over multi-tile airports                                                                                                                                                                                                                                                                                                          
  - Hangar location and direction tracking                                                                                                                                                                                                                                                                                                                              
                                                                                                                                                                                                                                                                                                                                                                        
  5. NewGRF Extensibility                                                                                                                                                                                                                                                                                                                                               
  - newgrf_airport.h/cpp - Custom airport specifications from NewGRF files                                                                                                                                                                                                                                                                                              
  - newgrf_airporttiles.h/cpp - Custom tile graphics and animations                                                                                                                                                                                                                                                                                                     
  - AirportSpec defines properties: size, noise, catchment, maintenance cost                                                                                                                                                                                                                                                                                            
  - Allows community-created custom airports                                                                                                                                                                                                                                                                                                                            
                                                                                                                                                                                                                                                                                                                                                                        
  6. User Interface                                                                                                                                                                                                                                                                                                                                                     
  - airport_gui.cpp - Airport building toolbar and picker window                                                                                                                                                                                                                                                                                                        
  - widgets/airport_widget.h - UI widget definitions                                                                                                                                                                                                                                                                                                                    
  - Class dropdown, layout selector, preview sprite display                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                                        
  7. Commands & Construction                                                                                                                                                                                                                                                                                                                                            
  - station_cmd.cpp - CmdBuildAirport implementation                                                                                                                                                                                                                                                                                                                    
  - Validates placement, handles rotations, calculates costs                                                                                                                                                                                                                                                                                                            
  - Town noise calculations and rating impacts                                                                                                                                                                                                                                                                                                                          
  - CmdOpenCloseAirport - toggle operational status                                                                                                                                                                                                                                                                                                                     
                                                                                                                                                                                                                                                                                                                                                                        
  Key Mechanisms                                                                                                                                                                                                                                                                                                                                                        
                                                                                                                                                                                                                                                                                                                                                                        
  Block System:                                                                                                                                                                                                                                                                                                                                                         
  - Prevents multiple aircraft from occupying the same space                                                                                                                                                                                                                                                                                                            
  - 64-bit bitmask in Airport::blocks                                                                                                                                                                                                                                                                                                                                   
  - Runways, taxiways, terminals each have reserved blocks                                                                                                                                                                                                                                                                                                              
  - Aircraft check/set blocks during movement                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                                                        
  Terminal Groups:                                                                                                                                                                                                                                                                                                                                                      
  - TERMGROUP - dynamic assignment of aircraft to available gates                                                                                                                                                                                                                                                                                                       
  - Aircraft request free terminal/helipad from group                                                                                                                                                                                                                                                                                                                   
  - Allows flexible multi-terminal operations                                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                                                        
  Movement Flags:                                                                                                                                                                                                                                                                                                                                                       
  - Control aircraft behavior: NoSpeedClamp, Takeoff, Land, Brake, HeliRaise, HeliLower, Hold                                                                                                                                                                                                                                                                           
  - Define speed limits and vertical movement for helicopters                                                                                                                                                                                                                                                                                                           
                                                                                                                                                                                                                                                                                                                                                                        
  Script API:                                                                                                                                                                                                                                                                                                                                                           
  - script/api/script_airport.hpp/cpp - AI and GameScript interface                                                                                                                                                                                                                                                                                                     
  - Allows AI players to query airport properties and build airports programmatically                                                                                                                                                                                                                                                                                   
                                                                                                                                                                                                                                                                                                                                                                        
  Data Flow                                                                                                                                                                                                                                                                                                                                                             
                                                                                                                                                                                                                                                                                                                                                                        
  1. Player selects airport type in GUI → airport_gui.cpp                                                                                                                                                                                                                                                                                                               
  2. Placement validated → CmdBuildAirport in station_cmd.cpp                                                                                                                                                                                                                                                                                                           
  3. Airport struct created with FSM reference                                                                                                                                                                                                                                                                                                                          
  4. Aircraft approaches → enters FSM at entry point                                                                                                                                                                                                                                                                                                                    
  5. State handlers move aircraft through positions                                                                                                                                                                                                                                                                                                                     
  6. Block system prevents collisions                                                                                                                                                                                                                                                                                                                                   
  7. Aircraft reaches terminal/hangar and completes journey
